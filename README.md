# 🎮 КиберэдэН: Цифровая эра

> Симулятор реальной жизни в эстетике кибергот. Учись жить — в игре и в реальности.

---

## Концепция

**Жанр:** Симуляция жизни / Кибергот / RPG-прогрессия  
**Движок:** Unreal Engine 5  
**Перспектива:** 1-е / 3-е лицо  
**Платформа:** PC (Windows / Linux / macOS)

Игрок создаёт Кибера — цифровую сущность — и встраивается в экосистему nazrOS. Главная цель не победить врага, а развиться: освоить реальные навыки, поднять ранг, стать частью кибер-цивилизации.

---

## Ключевые механики

- **Потребности персонажа** — сон, еда, гигиена, уборка влияют на настроение, репутацию и скорость прокачки
- **Сайд-квесты** — реальные навыки: стирка, посуда, заказ еды, документы, программирование
- **Основной сценарий: КАРМА** — необязателен, но влияет на мир
- **Прогрессия по рангам:** НАБЛЮДАТЕЛЬ → ОПЕРАТОР → АРХИТЕКТОР ЯДРА → ГЛАВНЫЙ РАЗРАБОТЧИК

## Структура репозитория

```
cybereden-game/
├── Docs/               # GDD, сценарий, концепт-арт
│   ├── GDD/            # Game Design Documents
│   ├── Scenario/       # Сценарий Кармы и сайд-квесты
│   └── Art/            # Концепт-арт, moodboard
├── Source/             # Исходный код (UE5 C++ / Blueprint)
│   ├── Core/           # Ядро: персонаж, потребности, прогрессия
│   ├── Quests/         # Система квестов
│   ├── NPC/            # NPC и диалоги
│   └── UI/             # Интерфейс
├── Content/            # Ассеты UE5 (не в git — через Git LFS)
│   ├── Meshes/
│   ├── Textures/
│   ├── Sounds/
│   └── Blueprints/
├── Config/             # Конфигурационные файлы UE5
├── Plugins/            # Плагины UE5
└── Tools/              # Вспомогательные скрипты
```
